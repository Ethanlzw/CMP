This folder consists of basic MATLAB codes for linear FEA, designed for teaching purpose in course "Computational Multiphysics" at University of Southern Denmark (SDU).
@Copyright belongs to Dr. Pei Li (lipei@sdu.dk) and SDU. This package of codes should not be used in any way for commercial purposes.

The following files in this code package are provided only templates and need to be completed by student:
1.  getElasticMatrix2D.m
2.  Edge2_ThermoMech.m
3.  assembleNatureBC_ThermoMech.m
4.  assmbleSystem.m
5.  Quad4_ThermoMech.m
6.  post_Quad4_ThermoMech.m
7.  solveSystem.m

Of the seven files, file 1, 4, 7 are exactly the same as what we instructed in the class for other field problems. If you have already completed these files, just copy them here and replace the template files.
